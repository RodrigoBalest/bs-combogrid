# Bootstrap Combogrid

JQuery Combogrid input for Twitter Bootstrap v3.x

## Getting Started

In your web page:

```html
<script src="jquery.js"></script>
<script src="dist/bs-combogrid.min.js"></script>
<link rel="stylesheet" href="dist/jquery.bs-combogrid.min.css">
<script>
jQuery(function($) {
  $('input').bs_combogrid();
});
</script>
```
